
# Todo

Simple todo app, with React+Vite and Typescript.

## Demonstration

![Project running](/project-images/todo.gif)


## Running local

Clone project

```bash
  git clone https://github.com/Bilusca/todo-rocketseat.git
```

Install dependecies

```bash
  yarn
```

Running app

```bash
  yarn dev
```

## Stack

**Front-end:** React, Vite, Sass.
